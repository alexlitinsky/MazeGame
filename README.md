# MazeGame
- Visualizes a maze using kruskal's algorithm
- Use BFS & BFS to traverse the nodes of the maze
- Implemented using the javalib library

# How to run
- Open the main file under your favorite java IDE
- run class "Maze"
- press 'd' to run DFS
- press 'b' to run BFS
- press 'r' to reset the maze
